## The application of Visual Knowledge Retrieval
We perform statistical analysis to set the retrieval thresh-
old, selecting all segments where every character exceeds
the threshold as the retrieval results. Then, we dynamically
expand the results based on predefined rules.
The relevant code can be found in the Retrival folder.
